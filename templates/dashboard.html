Installation Guide

Use the following commands to install packages from this repository:

Install a specific package:

pip install --extra-index-url {{ request.url_root.rstrip('/') }} package-name

Install with specific version:

pip install --extra-index-url {{ request.url_root.rstrip('/') }} package-name==1.0.0

Permanent configuration:

pip config set global.extra-index-url {{ request.url_root.rstrip('/') }}/

Example installation:

pip install --extra-index-url {{ request.url_root.rstrip('/') }} payo-cli
